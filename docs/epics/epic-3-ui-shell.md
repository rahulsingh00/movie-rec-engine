# Epic 3: Responsive Web UI

Create an interactive and gorgeous user interface to showcase movie listings, highlight similarity scores, search movies, and visualize NLP components.

## Acceptance Criteria

- [ ] Responsive web layout (sidebar/header + main content) utilizing grid and flexbox.
- [ ] Implement movie list search with autocomplete.
- [ ] Detail view for the selected movie showing its tagline, overview, genres, and a visual similarity bar.
- [ ] Recommendation grid showcasing the matching titles, their similarity score percentages, and matching genres.
- [ ] Modern UI details: dark glassmorphism styling, clean animations, and responsive breakpoints.
